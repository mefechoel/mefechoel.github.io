Recipe App was developed as part of my bachelor thesis. It consists of a NodeJS backend written in TypeScript using the server library Koa. It is responsible for providing the recipe data and running search queries. To access and view the recipes, one of three frontend apps can be used. Each frontend provides the same user interface and functionality, but is implemented using a different JavaScript framework, to showcase each frameworks advantages and disadvantages. The frameworks used are React, Vue and Svelte, all of which are reactive frontend frameworks.

Take a look at the page about my bachelor thesis for more information.
