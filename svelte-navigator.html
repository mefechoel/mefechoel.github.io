Bachelor Thesis

Svelte Navigator is an accessible single page app routing library for apps built with the Svelte frontend framework. When working on the Svelte frontend for my bachelor thesis, I was unable to find a routing libraries that worked correctly with my app. The main problem was, that the Svelte app was served from a subdirectory on my server, which existing libraries did not handle well.
Furthermore a lot of care was taken to ensure, that Svelte Navigator helps people build accessible websites. Historically, SPA routing libraries did not work well with assistive technologies. Svelte Navigator ensures that after a navigation, focus is moved to the part of the app that has actually changed, giving a starting point for users of assistive technologies to quickly discover added content.

Checkout Svelte Navigator's GitHub repository and its NPM package page.
